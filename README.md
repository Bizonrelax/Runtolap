# Runtolap

The repository contains an independent approach and unique analytical materials on the Lonely Runner Conjecture. 

*Status: This proposed hypothesis/proof is independent and has not yet undergone peer review or official verification by the academic authorities.*

## About the Method
The proof is built upon:
1. **LCM Synchronization:** Reducing the general continuous case to a subset of runners starting simultaneously at time fractions.
2. **Discrete Torus Dynamics:** Analyzing runner phases through discrete angular steps and applying a combinatorial approach (Dirichlet's principle) to resolve the "special case" where all periods divide the main runner's period.

### Суть метода
Доказательство опирается на:
1. **Синхронизацию через НОК:** Редукцию общего непрерывного случая к подмножеству бегунов, одновременно находящихся на старте в дискретные моменты времени.
2. **Динамику на дискретном торе:** Анализ фаз бегунов через дискретные шаги на окружности и применение принципа Дирихле для разрешения «специального случая», когда все периоды делят период основного бегуна.
